# Swiggy Sales Analysis Dashboard (Excel)
## Project Overview
This project presents an interactive Excel dashboard built using Swiggy sales data. It provides insights into sales performance, customer ratings, restaurant performance, regional trends, and food category analysis using Pivot Tables, Pivot Charts, Slicers, and Excel Dashboard features.
---
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Conditional Formatting
- Maps
- Treemap
- Doughnut Chart
---
## Dashboard Highlights
- Total Sales
- Total Orders
- Average Order Value
- Average Rating
- Rating Count
- Monthly Sales Trend
- Daily Sales Trend
- Regional Sales Analysis
- Top 10 Restaurants
- Sales by Food Type
- Top 5 Cities
- Quarter-wise Sales, Orders & Ratings
---
## Interactive Filters
- Month
- Food Type
- State
- Category
---
## Key Insights
- Identified monthly sales trends.
- Compared regional sales performance.
- Analyzed restaurant-wise sales.
- Evaluated food category contribution.
- Monitored customer ratings and order volume.
- Built an interactive dashboard using slicers.
